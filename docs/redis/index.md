`amphp/redis` is a non-blocking [Redis](http://redis.io) client for use with the [Amp](http://amphp.org/docs/amp/) concurrency framework.

**Required PHP Version**

- PHP 5.6+

**Installation**

```plain
$ composer require amphp/redis
```
