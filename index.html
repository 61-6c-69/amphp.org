Build fast, concurrent applications with PHP

AMPHP is a collection of event-driven libraries for PHP
designed with fibers and concurrency in mind.

Our wide range of packages allows building entire applications using non-blocking I/O to allow for concurrency in long-running processes and server applications. You can also leverage our libraries in portions of your existing application without a full-blown migration to non-blocking I/O.
