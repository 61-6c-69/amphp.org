amphp/amp is a non-blocking concurrency framework for PHP applications. 

<?php

function tick() {
    echo "tick\n";
}

echo "before run()\n";

Amp\run(function() {
    Amp\repeat("tick", $msInterval = 1000);
    Amp\once("Amp\stop", $msDelay = 5000);
});

echo "after run()\n";








    You can find the documentation in the amp guide.
